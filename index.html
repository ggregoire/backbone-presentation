Qu'est-ce-que Backbone.JS ?

Framework Javascript

basé sur le paradigme MVC

Créé par Jeremy Ashkenas pour son application single-page DocumentCloud Permet d'organiser et structurer son code Javascript

Mature et populaire

Un des plus anciens frameworks JS MVC

Grosse communauté derrière lui : 7eme projet sur Github

Les avantages pour notre JS

•Imposer un Architectural design pattern

•Séparer les données et le DOM

•Simplifier la persistance des données

•Données sous formes de modèles et de collections

•Découpage d'une page en plusieurs vues indépendantes

•Synchronisation automatique entre serveur/modèles/collections/DOM

Les avantages pour notre JS

•Events liés à l'interface et events liés aux données

•Code modulaire très simplement

•Support des routes via le hash de l'url (#)

•Fonctions + moteur de template livrés avec Underscore.js

•On garde jQuery pour manipuler le DOM

Un model a un getter et un setter

Backbone.Collection

Structure très proche du model

Backbone.Sync

Aucune requête AJAX à la main, et 1 seule URL
